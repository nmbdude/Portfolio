Man V Machine

Overview

Man V Machine is a platform shooter game where the player must defeat all the robots in the level to win. The player has a limited amount of health. Health can be restored by med-kits, which are dropped by robots when killed.

My contributions were the level editor which is based in windows forms and C#, I also helped design the game and implemented the player health. The game itself was made in the monogame framework.

Technologies

  • JavaScript
  • PixiJS graphics framework

Features

  • Grid generation system
  • Card matching system
  • Timer based on delta time
  • Highscore/score keeping system
