Man V Machine

Summary

  • Platform shooter where the player must defeat all robots in a level to win
  • Player has limited health, which can be restored using med-kits, which are dropped by robots
  • Built in C# using the MonoGame framework

Main Contributions

  • Developed the level editor using Windows Forms and C#
  • Helped design core gameplay mechanics and health systems
  • Implemented player health systems and med-kit mechanics

Build Breakdown

  • MonoGame: Used for game logic and rendering
  • C#: MonoGame's primary programming language
  • Winforms: Used for creating a separate level editor application that runs natively on Windows
