Fruit Slicer

Overview

Fruit slicer is a simplified version of the popular game Fruit Ninja. The player is tasked with slicing fruit as it is thrown into the air, and they must slice the fruit before it falls off the screen. If they miss a fruit, they lose all of their points. If they slice a bomb, they lose the game. Once they get to a score of 10, they win. The input system and game mechanics are all programmed by me in C#.

Technologies

  • Unity Game Engine
  • C#
  • Github

Features

  • Scripted Implementation of Unity's Physics
  • User interactivity
  • Mouse Swipe Detection System
