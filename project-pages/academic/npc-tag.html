NPC Tag

Overview

NPC Tag is a simple interactive experience meant to demonstrate the different basic behaviors that NPCs can exhibit when moving around in a game. In this project, the Flee, Seek, Wander, and Obstacle Avoidance are implemented. While the implementation in this project isn't a perfect one, the behaviors are still recognizable and can be seen in the NPCs as they move around the screen. The user can interact with the world via the spacebar and the mouse buttons. The project was developed in Unity, my contribution to this project was the NPC behaviors and the implementation of the world context.
