Memory Blast

Overview

Memory Blast is an online card matching game. The player must match all the cards before time runs out. Every time that they match a card, a few seconds of time is added to their timer. If they get a match incorrect, there is a brief period where they cannot match any cards. The game will keep track of highest score (that isn't a win) and store it in local storage for later play sessions. The games functionality and the shapes to match were made by me, from scratch and based in the PixiJS graphics framework.

Technologies

  • JavaScript
  • PixiJS graphics framework

Features

  • Grid generation system
  • Card matching system
  • Timer based on delta time
  • Highscore/score keeping system
