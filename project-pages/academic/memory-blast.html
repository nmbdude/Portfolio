Summary

A casual, grid-based card memory matching game built with PixiJS. Includes a delta-time timer, score keeping, high-score saving, and randomized card layout. Higher difficulties increase grid size and card variety. Built entirely from scratch including art assets. Future work: main menu, more art, difficulty options.

Build Breakdown

  • Framework: PixiJS
  • Language: JavaScript
  • Systems: Grid generation, matching logic, delta-time timer, scoring/high-score
