Crazy Taxi

Overview

Crazy Taxi is a simple waypoint collection game, where the player controls a vehicle and has to reach a waypoint before the time runs out. The player can collect more time by reaching the waypoint, and the object above the taxi will point to the nearest waypoint. The game was created in Unity using C#, and has a custom physics system for the vehicle, where it will always snap to the surface of the terrain, which is generated from a perlin noise algorithm. My contribution to the project was the physics, waypoint, and terrain generation systems, as well as all the game logic.
