The House Sitter

Overview

The House Sitter is my main personal project. I started working on it a few years ago, however due to an issue with the project files, I had to start over from scratch. Being developed in Unreal Engine 5, it's a long term project that I plan to work on for the next few years in my free time. The House Sitter is entirely programmed by me in C++, and I've been working on the art assets as well, however, those take longer for me to make than the code. The project leverages an updated version of an inventory system that I developed for The Lofi Cafe.

Technologies

  • Unreal Engine 5
  • C++
  • Blender
  • Unreal Engine Blueprints
  • Microsoft Azure

Features

  • Implementation of different post-processing effects
  • User input
  • Custom Inventory System
  • Fully Immersive Environment
